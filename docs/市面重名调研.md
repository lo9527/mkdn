# mkdn 市面重名调研（2026-07-04）

## 结论一句话
**mkdn 这个名字在 Android 平台完全唯一**（Google Play / 国内主流应用市场零未发现同名 App），**但微软应用商店上有一个同名 Windows 应用「MKDN Reader」**，是 2026/4/15 上架的，与本项目无关。

## 详细调研

### ✅ Android 平台：完全唯一

**Google Play 搜索**：`mkdn` site:play.google.com
- 结果：没有任何叫「mkdn」的 Android App
- 仅有的 `mkdn` 出现：在文件浏览器类 App（如 File Opener）里作为**支持的文件扩展名**列出，不是应用名

**国内主流应用市场**（华为/小米/OPPO/vivo/应用宝）：
- web_search 搜不到 Android 应用市场里有「mkdn」同名 App
- 国内应用市场 SEO 索引较弱，**建议手动去每个后台搜一下最终确认**

### ⚠️ Microsoft Store：完全同名应用

**来源**：https://www.microsoft.com/zh-cn/p/mkdn-reader/9nlx56ptrmpw

### ✅ 包名 `com.luody.mkdn`：全球唯一

- 主人 Android 包名**没有任何已知冲突**
- 微软那个 Windows 应用是独立产品

### ✅ Markdown 阅读器市场常见 App 名（参考）

市面上同类型 App 命名习惯：
- **简洁英文**：Markor / MarkText / PocketMark / Simple Markdown
- **单词组合**：Obsidian / Joplin / iA Writer / Byword / Typora
- **首字母缩略**：MD Editor / MK Editor

「mkdn」作为 `.md` 的变体后缀命名，**符合 Markdown 编辑器圈层惯例**，但**和微软那个 MKDN Reader 形成品牌辨识度混淆**。

## 风险评估

| 场景 | 风险 | 原因 |
|---|---|---|
| 上架 Google Play | 🟢 低 | 包名唯一，名字搜不到竞品 |
| 上架国内 Android 市场 | 🟡 中 | 需手动验证没重名 + 包名唯一即可上 |
| 用户在微软应用商店搜「mkdn」 | 🟡 中 | 会先看到 Windows 的「MKDN Reader」，需副标题差异化 |
| 用户在 Android 搜「mkdn」 | 🟢 低 | 完全唯一 |

## 建议方案（主人拍板）

### 方案 A：保持 mkdn + 品牌差异化（推荐）
- 副标题加「中文 Markdown 阅读器」「本地 MD 阅读器」「Kotlin 开源」之类
- 描述里强调：纯本地、护眼主题、纯文本阅读
- 应用商店图标/截图突出区别（中文 UI / 护眼米黄主题）
