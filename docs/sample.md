> 引用块：可以放一些引用的话
> 
> 多行引用会一起显示

## 四、阅读建议

mkdn 默认为 **护眼米黄主题** + **18sp 字号**。建议：
1. 在顶栏右侧调整字号和主题
2. 选中文字 → 顶部弹出的 🔖 按钮可标重点
3. 点击顶栏 📑 打开目录大纲
4. 点击 ✏️ 进入简单编辑模式
